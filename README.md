The **Drum Kit / Sound Generator** is a **web-based interactive project** that allows users to play drum sounds in a fun and engaging way.  

---

## About the Project

- Users can **click on drum buttons** displayed on the webpage to play different drum sounds.
- Users can also **press specific keyboard keys** (`w`, `a`, `s`, `d`, `j`, `k`, `l`) to trigger the corresponding drum sounds.
- The project simulates a **real drum kit**, making it an entertaining tool for creating beats.
- Built using **HTML, CSS, and JavaScript**, leveraging event listeners and the Audio API to produce sounds.

---

## Key Features

- Interactive and responsive drum buttons.
- Multiple drum sounds: toms, snare, crash, and kick.
- Real-time audio playback with both mouse and keyboard input.
- Easy to expand with additional sounds or animations.

---

This project is ideal for anyone looking to **learn JavaScript event handling, audio playback, and interactive web design**, while having a fun, hands-on experience with sound generation.
